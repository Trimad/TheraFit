Routes

Endpoint Method Description
/data GET Fetch all data
/data/clients GET Fetch clients data and apply optional search filter
/data/therapists GET Fetch therapists data and apply optional search filter
/data/update GET Fetch updated data

GET /data

Returns all data as a JSON object.

GET /data/clients

Returns filtered clients and all therapists as a JSON object. The clients can be filtered using an optional query parameter "search".

Example: /data/clients?search=john

GET /data/therapists

Returns filtered therapists and all clients as a JSON object. The therapists can be filtered using an optional query parameter "search".

Example: /data/therapists?search=doe

GET /data/update

Returns updated data as a JSON object.
